关于画图

  • uml类图, startUML 看起来还不错,但是我想画的并不是标准的类图,而是随心所欲的数据结构图, https://www.cnblogs.com/shindo/p/5579191.html
  • process on,在线做图,傻瓜式,但是功能少
  • draw.io,目前发现的最适合我的画图软件,支持和github帐号绑定,从github sync非常赞,创建的时候选择 软件（程序员）, 这里可以下载draw.io的离线版本. linux的离线版本可能有点问题，使用minphone的方法，先将star-uml的tar.gz包解压，然后将window下exe安装后的
    C:\Program Files\draw.io\resources\ 下的 app.asar + app-update.yml 替换到 start-uml目录下的resource下，运行startuml即可.（貌似破解的话还需要修改asar包中的内容，涉及到解包和打包操作，百度start uml打包拆包）

关于 markdown

  • 官网 http://www.markdown.cn/
  • [markdown在线编辑，开源的，可以和github账户绑定,但是用起来一般] https://stackedit.io/app#
  • [在线markdown转pdf,貌似还有一些bug,比如todolist无法正常显示] http://www.mdtr2pdf.com/index_en.html
  • 目前使用觉得最舒服的还是 vnote, github开源,编辑方便,支持转pdf~

关于 vscode

  • 看起来vscode不适合我,linux下一直在后台加载更新东西,奇慢
    安装：
    下载 code_1.28.1-1539281690_amd64.deb
    sudo dpkg -i .deb
    sudo apt-get install -f # Install dependencies

关闭自动更新：
file->preference->settings，点击 open settings.json,搜索 channel,将 update.channel 改为 none，并将 extensions.autoUpdate 改为 false即可
{
“update.channel”: “none”,
“workbench.colorTheme”: “Quiet Light”,
“extensions.autoUpdate”: false
}

去掉 Render Indent Guides 前面的勾

常用快捷键（file->preference->keyboard shortcuts 可以定制）
文件内部查找直接ctrl+f，然后 f3或f4查找上一个或下一个
查找所有引用：vscode左侧table的放大镜，然后输入查找，完成后shift+f3 或 shift+f4查找上一个或下一个位置
打开当前文件 ctrl +f4
跳转到函数定义：f12(返回alt+,)

按文件名打开某文件 ctrl + p
调出本文件内函数/结构体 定义 ctrl+shift+t
全局查找某个symbol（比如函数，结构体） ctrl+t
